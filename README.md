# My Budget Vue

Vue 3 приложение для управления бюджетом.

## Технологии

- Vue 3 (Composition API)
- TypeScript
- Vite
- Vue Router
- Pinia
- TanStack Query Vue
- Supabase
- vue-i18next
- Tailwind CSS

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Настройка окружения

Скопируйте `.env.example` в `.env` и заполните переменные окружения для Supabase:

```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```
